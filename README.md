nodeJS + express
Приложение для визуализации git-репозитория.

### Установка
```sh
$ git clone https://github.com/voronin-ivan/shri-4-nodejs.git
$ npm install
$ npm run clone
```

### Запуск (development)
```sh
$ npm run dev
```
